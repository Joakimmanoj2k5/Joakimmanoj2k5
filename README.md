## About Me

I am Joakim Manoj, a B.Tech Information Technology student at Government Engineering College, Barton Hill, Kerala. I enjoy building practical software that sits at the intersection of AI, computer vision, and modern web development.

My current focus is on:

- AI and machine learning projects with real-world use cases
- Computer vision systems and gesture-based interfaces
- Full stack applications with clean product thinking
- Sharpening problem solving through continuous building

## Featured Projects

### [Mudra](https://github.com/Joakimmanoj2k5/mudra)
Real-time Indian Sign Language recognition system built with computer vision and deep learning. This project reflects my interest in accessible AI and intelligent interaction.

**Stack:** `Python` `TensorFlow` `MediaPipe` `OpenCV`

### [StudySync AI](https://github.com/Joakimmanoj2k5/StudySync-AI)
An AI-powered study companion focused on improving how students organize, revise, and retain information through a smarter workflow.

**Stack:** `TypeScript` `Next.js` `Vercel`

### [Project Manager](https://github.com/Joakimmanoj2k5/Project-manager)
A project management application designed for team coordination, task tracking, and structured progress monitoring.

**Stack:** `PHP` `MySQL` `HTML` `CSS`

### [Portfolio Website](https://github.com/Joakimmanoj2k5/Joakimmanoj2k5.github.io)
My personal portfolio site, built to present projects, skills, and the kind of work I want to do more of.

**Stack:** `HTML` `CSS` `JavaScript`
